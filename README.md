# uiflows-convert-png

## command

```
ucp {filepath}
```

## Usage

### 1.prepare any uiflows-code-file

```uiflows:./sample/uiflows.md
[what the user sees]
what they do
==> what they see next
what they do
==> what they see next2

[what they see next]
what they do next

[what they see next2]
```

### 2.`$ ncp ./sample/uiflows.md`

### 3.created `./{timestamp}.png`
